    5.1. Franklin's Markdown

    As usual with sites built with Franklin.jl, you may easily insert math and execute code when desired.

    Math

    \[ \exp(i\pi) + 1 = 0 \]

    Code

    println("hello world!")
    hello world!

    Plot

    using Plots
    x = 0.0:0.01:2pi
y = sin.(2x) + sin.(5x)
plot(x, y)
