    • Welcome

    This website introduces and demonstrates an adaptation to Franklin.jl of the CSS layout used in the package Books.jl, at version 1.2.2, with lots of other goodies, such as automatic generation of the table of contents; automatic numbering of sections and navigation links; pages written in either Markdown, Jupyter notebook or julia script and rendered/executed via Franklin.jl, Weave.jl or Literate.jl; automatic generation of Jupyter notebooks; and badges on each page to download or open the associated notebooks in NBViewer or Binder and for viewing the source code.
