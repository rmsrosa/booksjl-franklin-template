    • Welcome

    This website introduces and demonstrates an adaptation to Franklin.jl of the CSS layout used in the package Books.jl, at version 1.2.2, with lots of other goodies, such as automatic generation of the table of contents, automatic numbering of sections, navigation links, automatic rendering of julia scripts and fenced markdowns via Weave.jl or Literate.jl, etc.
